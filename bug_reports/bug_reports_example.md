# Баг-репорт: Не добавляется товар в корзину с мобильного устройства

## Основная информация
- ID бага: BUG-001
- Приоритет: High (Высокий)
- Серьезность: Major (Критический)
- Статус: New (Новый)
- Окружение: Production (Продакшен)
- Дата создания: 20.01.2026
- Автор: Анна

## Описание
На мобильной версии сайта кнопка "В корзину" на странице товара не реагирует на нажатие. Пользователь не может добавить товар в корзину с мобильного устройства.

## Шаги воспроизведения
1. Открыть сайт example.com на iPhone (iOS 19.0)
2. Перейти в любой раздел товаров (например, "Смартфоны")
3. Открыть карточку любого товара
4. Нажать кнопку "В корзину" (оранжевая кнопка под описанием товара)

## Фактический результат
Кнопка нажимается (меняет цвет при тапе), но:
- Товар не добавляется в корзину
- Не появляется сообщение об успешном добавлении
- Счетчик товаров в корзине (в шапке сайта) не изменяется
- В консоли браузера появляется ошибка: TypeError: Cannot read properties of undefined (reading 'addToCart')

## Ожидаемый результат
После нажатия кнопки "В корзину":
1. Товар добавляется в корзину
2. Появляется всплывающее уведомление "Товар добавлен в корзину"
3. Счетчик товаров в корзине увеличивается на 1
4. Пользователь может перейти в корзину и увидеть там добавленный товар

## Дополнительная информация
### Консольные ошибки:
Uncaught TypeError: Cannot read properties of undefined (reading 'addToCart')
at HTMLButtonElement.onclick (product.js:45)
at mobile-handler.js:23

### Влияние на бизнес:
- Пользователи не могут совершать покупки с мобильных устройств
- Потеря конверсии (более 60% трафика приходит с мобильных)
- Отрицательный пользовательский опыт

### Условия воспроизведения:
- Устройства: iPhone 12/13/14/15/16, Samsung Galaxy S21/S22
- Браузеры: Safari, Chrome (мобильные версии)
- Версия ОС: iOS 19+, Android 11+
- Частота: 100% (воспроизводится всегда)

### Рабочее окружение:
- Устройство: iPhone 15
- ОС: iOS 19.4.1
- Браузер: Safari 19.4
- Разрешение экрана: 393x852
- Сеть: Wi-Fi / 5G

1. На десктопной версии баг не воспроизводится
2. Проблема только с кнопкой "В корзину" на странице товара
3. Другие кнопки на сайте работают корректно
4. Баг был обнаружен при smoke-тестировании мобильной версии
