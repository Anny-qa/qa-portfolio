# Тест-кейсы: Поиск товаров в интернет-магазине

## Общая информация
- Проект: Интернет-магазин "TechStore"
- Модуль: Поиск товаров
- Дата создания: 20.01.2026
- Автор: Анна

## Тест-кейсы

### TS-001: Поиск существующего товара
Предусловия: Открыта главная страница сайта
Шаги:
1. В поле поиска ввести "iPhone 13"
2. Нажать кнопку "Найти" или клавишу Enter
Ожидаемый результат: Отображаются товары, содержащие "iPhone 13" в названии или описании
Статус: ✅ PASS

### TS-002: Поиск несуществующего товара
Предусловия: Открыта главная страница сайта
Шаги:
1. В поле поиска ввести "абвгдеёжз"
2. Нажать кнопку "Найти"
Ожидаемый результат: Сообщение "Товары не найдены"
Статус: ✅ PASS

### TS-003: Поиск с пустым запросом
Предусловия: Открыта главная страница сайта
Шаги:
1. Оставить поле поиска пустым
2. Нажать кнопку "Найти"
Ожидаемый результат: Кнопка "Найти" неактивна или появляется подсказка "Введите запрос"
Статус: ✅ PASS

### TS-004: Поиск с спецсимволами
Предусловия: Открыта главная страница сайта
Шаги:
1. В поле поиска ввести "!@#$%^&*"
2. Нажать кнопку "Найти"
Ожидаемый результат: Сообщение "Некорректный запрос" или "Товары не найдены"
Статус: ✅ PASS

### TS-005: Поиск с учетом регистра
Предусловия: Открыта главная страница сайта
Шаги:
1. В поле поиска ввести "IPHONE"
2. Нажать кнопку "Найти"
Ожидаемый результат: Отображаются товары "iPhone" (поиск нечувствителен к регистру)
Статус: ✅ PASS

### TS-006: Поиск по частичному совпадению
Предусловия: Открыта главная страница сайта
Шаги:
1. В поле поиска ввести "iph"
2. Нажать кнопку "Найти"
Ожидаемый результат: Отображаются товары, содержащие "iph" в названии (iPhone, Philips и т.д.)
Статус: ✅ PASS

### TS-007: Поиск товара с пробелами в начале/конце
Предусловия: Открыта главная страница сайта
Шаги:
1. В поле поиска ввести "  iPhone 13  "
2. Нажать кнопку "Найти"
Ожидаемый результат: Пробелы обрезаются, отображаются товары "iPhone 13"
Статус: ✅ PASS

### TS-008: Поиск с использованием выпадающих подсказок
Предусловия: Открыта главная страница сайта
Шаги:
1. В поле поиска начать вводить "sams"
2. Дождаться появления выпадающего списка с подсказками
3. Выбрать подсказку "Samsung Galaxy S23"
Ожидаемый результат: Отображаются товары Samsung Galaxy S23
Статус: ✅ PASS

### TS-009: Очистка поля поиска
Предусловия: В поле поиска введен текст "iPhone"
Шаги:
1. Нажать на крестик в правой части поля поиска
Ожидаемый результат: Поле очищается, поиск не выполняется
Статус: ✅ PASS

### TS-010: Поиск товара по артикулу
Предусловия: Открыта главная страница сайта
Шаги:
1. В поле поиска ввести артикул товара "A12345"
2. Нажать кнопку "Найти"
Ожидаемый результат: Отображается конкретный товар с артикулом A12345
Статус: ✅ PASS

## Техники тест-дизайна, использованные в кейсах:
1. Классы эквивалентности: существующий/несуществующий товар
2. Граничные значения: пустой запрос, максимальная длина
3. Негативное тестирование: спецсимволы, некорректные данные
4. Позитивное тестирование: корректные запросы
5. Тестирование UI: работа выпадающих подсказок, кнопок
